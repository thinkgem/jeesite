## 介绍

jeesite-web-ai-cms 拷贝自 jeesite-web 只保留了 CMS 和 AI 模块的引用，

可直接运行 AiCmsApplication.java 文件，启动一个 Web 服务。

启动 Web 服务后，可通过网页浏览器访问 JeeSite 系统。
