## 介绍

jeesite-web-ai 拷贝自 jeesite-web 只保留了 AI 模块的引用，

可直接运行 AiApplication.java 文件，启动一个 Web 服务。

启动 Web 服务后，可通过网页浏览器访问 JeeSite 系统。
